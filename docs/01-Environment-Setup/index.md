# CATIA 환경 파일 만들기

이 장은 CATIA 교육 과정의 시작 단계로,  
**CATIA란 무엇인지 이해하고, 사용을 위한 기본 환경을 준비하는 것**을 목표로 합니다.

CATIA를 처음 접하는 사용자가  
프로그램의 개념, 화면 구성, 기본 조작 방식, 그리고  
실무 및 교육에 필요한 **환경 설정 방법**까지 단계적으로 학습할 수 있도록 구성되어 있습니다.

---

## 학습 목표

이 장을 완료하면 다음을 이해할 수 있습니다.

- CATIA V5의 기본 개념과 역할
- Feature 기반 설계와 파라메트릭 개념
- CATIA 사용자 화면의 구성 요소
- 기본 조작 방법과 화면 제어 방식
- CATIA 환경 파일의 구조와 설정 방법
- 사용자 정의 및 기본 옵션 설정 방법

---

## 구성 내용

이 장은 아래와 같은 순서로 구성되어 있습니다.

1. **소개**  
   CATIA 개요 및 교육 과정 설명

2. **CATIA 이해**  
   CATIA V5, Feature, Modeling 개념, 설계 의도

3. **CATIA 사용자 화면**  
   Workbench, Specification Tree, Toolbar, View 제어

4. **조작법**  
   마우스 사용법 및 기본 화면 조작

5. **CATIA 도움말 및 실습**  
   도움말 활용 방법과 기본 예제 실습

6. **환경 만들기**  
   CATIA 환경 파일 구조 및 실행 환경 설정

7. **기본 사용자 정의**  
   메뉴, Toolbar, 명령어 사용자 설정

8. **기본 옵션**  
   Display, Sketcher, Part, Assembly 등 주요 옵션

---

## 학습 안내

- 본 과정은 **기본 개념 → 화면 이해 → 환경 설정**의 흐름으로 진행됩니다.
- 실습은 실제 CATIA 화면을 기준으로 설명됩니다.
- 이후 장(Part, Assembly, Drafting 등)의 기초가 되는 내용이므로  
  **환경 설정 단계까지 반드시 이해하는 것을 권장**합니다.
