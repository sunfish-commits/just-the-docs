# Parametric

Feature는 제어 가능한 Dimension(치수)과 Relation(관계)을 가지고 있으며, 이러한 요소를 Parameter라고 합니다.

Parameter를 활용하면 설계 의도를 반영하여 모델링을 쉽고 빠르게 제어할 수 있습니다.

Parameter는 Dimension과 Relation으로 구성됩니다.
