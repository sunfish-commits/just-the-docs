### CATIA 문서 화면 전환

CATIA 내의 여러 문서를 Window 기능과 단축키를 이용해 손쉽게 전환할 수 있습니다.

① 문서 화면 분할  
Window > Tile Horizontally / Vertically / Cascade  
→ 여러 문서를 동시에 보거나 배치할 수 있습니다.

<img src="{{site.url}}/images/Switching-CATIA-Document-Screens/Switching-Document_001.png" alt="Switching-Document_001" style="max-width: 50%;" />

② 문서 선택  
열린 문서 중에서 원하는 문서를 선택하여 전환할 수 있습니다.

<img src="{{site.url}}/images/Switching-CATIA-Document-Screens/Switching-Document_002.png" alt="Switching-Document_002" style="max-width: 80%;" />

③ 단축키 화면 전환  
"Ctrl + Tab"을 누르면 열린 문서들이 순차적으로 전환됩니다.

<img src="{{site.url}}/images/Switching-CATIA-Document-Screens/Switching-Document_003.png" alt="Switching-Document_003" style="max-width: 80%;" />
