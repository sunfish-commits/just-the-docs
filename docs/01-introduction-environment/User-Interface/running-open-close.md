# CATIA 실행 및 열기 / 닫기

CATIA의 기본 실행, 열기, 종료 방식에 대해 설명합니다.


 ![running-open-close_001]({{ site.baseurl }}/images/running-open-close/running-open-close_001.png){: width="80%"}  


1. CATIA 실행 방법

   a. 시작 > 프로그램 > CATIA 메뉴로 부터 선택 하여 실행

   b. 윈도우 Desktop에서 CATIA 아이콘을 더블 클릭하여 실행

   c. 기존 CATIA 문서를 더블 클릭하여 실행

2. CATIA 열기

   a. 기존 파일 열기 (File > Open)

   b. 새로운 문서 파일 생성 (File > New)

   c. 기존 파일을 새로운 파일로 생성 (File > New From)

   d. CATIA 문서를 더블 클릭하여 파일 열기

   e. CATIA 문서를 CATIA의 화면에 Drag & Drop을 통해 열기

3. CATIA 닫기

   a. CATIA의 File > Close을 통해 문서 닫기

   b. CATIA 창(문서)의 “닫기 아이콘”을 통해 문서 닫기
