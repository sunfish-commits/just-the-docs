### 기본 환경 설명

CATIA의 사용자 화면은 모델을 생성·관리하기 위한 주요 요소들로 구성되어 있으며, 

각 요소는 모델링 작업을 수행하는 데 핵심적인 역할을 합니다.

![Basic-Environment-Overview_001]({{ site.baseurl }}/images/Basic-Environment-Overview/Basic-Environment-Overview_001.png)

CATIA는 Windows 기반 응용 프로그램과 유사하게 동작하며, 다양한 프로그램과 데이터를 연결해 사용할 수 있습니다. 

파일 이름 작성 시 영문 사용을 권장합니다.

**1. 기본 단축키(Windows 공통 기능)**

일반 메뉴, 표준 키보드, 윈도우에서 지원하는 단축키

- Ctrl+C, Ctrl+V, Ctrl+Z, Ctrl+Y, Ctrl+F, Ctrl+S 등

- Drag & Drop 기능 지원

![Basic-Environment-Overview_002]({{ site.baseurl }}/images/Basic-Environment-Overview/Basic-Environment-Overview_002.png)


**2. 응용 프로그램 간 데이터 공유(OLE 지원)**

Object Linking and Embedding을 통해 Excel 등의 데이터를  CATIA 모델과 연결할 수 있습니다.

![Basic-Environment-Overview_003]({{ site.baseurl }}/images/Basic-Environment-Overview/Basic-Environment-Overview_003.png)


**3. 한글 및 특수 문자 제한**

- 파일 이름 저장 시 한글 미지원 → 영문 사용 권장 
- ! @ # $ % ^ & * ( ) . , / 등의 특수문자는 사용을 권장하지 않음

![Basic-Environment-Overview_004]({{ site.baseurl }}/images/Basic-Environment-Overview/Basic-Environment-Overview_004.png)
