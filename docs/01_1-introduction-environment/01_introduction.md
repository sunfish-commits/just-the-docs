# CATIA V5 

CATIA는 프랑스 Dassault Systèmes에서 개발·판매하는 3D CAD & PLM 소프트웨어입니다.

다양한 산업군에서 사용되며, 차별화된 디지털 기술을 제공합니다.

CATIA는 설계 과정에서 발생하는 문제에 대한 다양한 솔루션과 방법을 제시합니다.
