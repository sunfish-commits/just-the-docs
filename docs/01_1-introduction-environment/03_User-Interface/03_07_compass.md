# Compass

1. Compass는 3D 모델(특히 Assembly Design 및 Digital Mockup Workbench)에서 
    물체를 물리적으로 이동·회전시키기 위해 사용하는 도구입니다.
2. Compass는 기본적으로 XY Plane을 기준으로 움직이며, 회전 및 이동 방향을 제어할 수 있습니다.
