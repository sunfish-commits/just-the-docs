# Normal View

- 선택한 요소(Face, Plane, Line, 2D Curve 등)에 대해 해당 요소의 법선 방향(Normal)으로 화면을 정렬하는 기능입니다.
- 주로 면(Face)을 선택한 뒤 Normal View를 적용하여 정면 기준 뷰를 설정할 때 사용합니다.
