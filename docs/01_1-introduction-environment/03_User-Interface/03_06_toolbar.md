# Toolbar

작업에 필요한 기능들을 쉽게 사용할 수 있도록 유사한 기능들을 묶어 제공하는 메뉴입니다.

1. View > Toolbar 메뉴 선택
2. 표시 중인 Toolbar는 체크(√)로 표시됨
3. 원하는 Toolbar를 선택해 표시하거나 숨길 수 있음
