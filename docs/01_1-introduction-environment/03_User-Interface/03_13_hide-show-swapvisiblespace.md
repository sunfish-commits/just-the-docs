# Hide/Show , Swap Visible Space

Hide/Show와 Swap Visible Space는 복잡한 모델에서 필요한 요소만 시각적으로 관리할 수 있도록 도와줍니다.


![hide-show-swapvisiblespace_001]({{ site.baseurl }}/images/01_1/03_13_hide-show-swapvisiblespace/hide-show-swapvisiblespace_001.png){: width="40%"}


1. Hide/Show
    - 요소들을 숨기거나 다시 나타나게 하는 기능입니다.
    - 오른쪽 마우스(Contextual Menu)를 통해 자주 사용됩니다.


    ![hide-show-swapvisiblespace_002]({{ site.baseurl }}/images/01_1/03_13_hide-show-swapvisiblespace/hide-show-swapvisiblespace_002.png){: width="50%"}

    - 또는 단축키 설정으로 사용합니다.


    ![hide-show-swapvisiblespace_003]({{ site.baseurl }}/images/01_1/03_13_hide-show-swapvisiblespace/hide-show-swapvisiblespace_003.png){: width="50%"}


2. Swap Visible Space
    숨겨진 요소들이 있는 공간(Hide Space)과 보이는 공간(Show Space)을 
    전환하는 기능입니다.
    
    ![hide-show-swapvisiblespace_004]({{ site.baseurl }}/images/01_1/03_13_hide-show-swapvisiblespace/hide-show-swapvisiblespace_004.png){: width="100%"}
