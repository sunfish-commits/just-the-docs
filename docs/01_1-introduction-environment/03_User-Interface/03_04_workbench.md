# Workbench

Part를 생성하거나 특정 작업을 하기 위해서는 적절한 Workbench로 전환해야 하며, 위와 같은 방법으로 이동할 수 있습니다.

1. 시작 메뉴에서 원하는 Workbench 선택
2. File > New에서 특정 문서 유형 선택 시 해당 Workbench 자동 연결 
3. 사용자 지정한 Workbench 아이콘을 통해 바로 진입
